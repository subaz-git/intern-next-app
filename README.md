## Questions
```
A page that stores questions that asked by several users
```
---

### Features
- [x] Upvote or Downvote
    - 1 Vote per question
    - 1 Vote per browser
- [x] AI enhancement for question
- [x] TimeStamp
- [x] Category
- [x] Search Questions
- [x] Sort Quesions
    - Modified (Old - New)
    - Modified (New - Old)
    - Alphabetical (A - Z)
    - Alphabetical (A - A)
- [ ] Live Updating Questions and Vote counts  

---

#### How to run

1. Clone the repo

```
git clone https://github.com/subaz-git/intern-next-app.git
```
2. Install Packages

```
npm install
```

3. Run server

```
npm run dev
```
